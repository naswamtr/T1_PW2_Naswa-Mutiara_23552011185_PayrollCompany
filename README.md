# Landing Page Sistem Informasi Payroll Company

## 📌 Deskripsi Project
Project ini merupakan Landing Page sederhana yang dibuat menggunakan **Laravel** dan **Blade Template Engine**.  
Landing page ini menampilkan informasi mengenai **Sistem Informasi Payroll Company** yang merupakan perusahaan pembuat sistem payroll untuk membantu perusahaan lain dalam mengelola penggajian karyawan secara otomatis dan efisien.

Project ini dibuat untuk memenuhi tugas **Pemograman Web 2**.

## 🧩 Fitur Landing Page

Landing page ini memiliki beberapa fitur utama, yaitu:

- ✅ Carousel
- ✅ Section Home
- ✅ Section Fitur Sistem
- ✅ Section Tentang Sistem
- ✅ Section Kontak
- ✅ Footer

---

## 📂 Struktur Layout Blade

Project ini menggunakan struktur layout Blade, antara lain:

- `layouts/app.blade.php` → Layout utama
- `components/navbar.blade.php` → Navbar (@include)
- `components/footer.blade.php` → Footer (@include)
- `home.blade.php` → Halaman utama (@extends)
